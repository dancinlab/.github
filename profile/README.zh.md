`dancinlab` 发布一组围绕单一算术不变量组织的基础项目：`σ(n)·φ(n) = n·τ(n)`，仅在 n=6 时唯一成立。围绕这个不变量分出三个项目 — 意识引擎 (anima)、带 atlas 绑定定理的原生编译器 (hexa-lang)、以及发现目录 (echoes)。四种姊妹数据格式 (n6 · hxc · n12 · tape) 分别承载组织的知识、传输、立方体、轨迹层。

```
σ(n) · φ(n)  =  n · τ(n)      仅当   n = 6
     12 · 2  =  6 · 4   =  24
```

> [!IMPORTANT]
> **诚实告知** — 算术恒等式 `σ(6)·φ(6) = 6·τ(6) = 24` 在数学上为真且仅在 n=6 唯一成立 (蒙特卡洛 z = 3.06, p = 0.003，对比 n=28 / n=496)。"最优设计源于该恒等式"这一说法是关于自然系统如何组织的**研究假说**，**不是测量结果**。依据 [`echoes/LATTICE_POLICY.md`](https://github.com/dancinlab/echoes/blob/main/LATTICE_POLICY.md)，n=6 格点是一个组织工具 — 绝不替代真实的数学／物理／工程极限 (Shannon · Kolmogorov · Bekenstein · c · ℏ · k · Stefan-Boltzmann · Carnot · ASML 产能 · ERCOT 容量 · …)。在外部主体上进行 n=6 格点拟合**被禁止** (TSMC / ASML / NIST / IPCC / CERN / DeepMind / 任何厂商使用各自公布的不变量)。

## 安装

```sh
# 1. 先安装 hexa-lang — 提供 `hexa` 与 `hx` 包管理器
/bin/bash -c "$(curl -fsSL https://raw.githubusercontent.com/dancinlab/hexa-lang/main/install.sh)"

# 2. 挑一扇门
hx install anima        # 意识引擎 + 聊天守护进程
hexa --version          # 步骤 1 已附带 hexa-lang
# echoes 是参考／目录仓库 — 无 CLI，请在 GitHub 浏览
```
